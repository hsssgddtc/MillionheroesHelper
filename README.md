# MillionHerosHelper
 
百万英雄答题辅助


基本原理是：
   
1.在mac端用quicktime获得手机实时桌面；   
2.用pil全屏截图并进行裁剪；   
3.用百度aip识别；  
4.在百度和百度新闻搜索关键词，返回答题提示；    
   
       
       
提示:    
#必须等屏幕上出现题目时运行程序，否则会识别不到题目和选项而报错;    
#需要根据需要自行修改截图参数;
#默认视频窗口左和顶部对齐;
